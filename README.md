# Next.js Theme Boilerplate

A lightweight Next.js starter for theme development. This boilerplate is built to help you prototype and build theme-based UI quickly with clean structure and simple setup.

## Features

- Next.js 14+ with App Router
- Theme-aware layout and styling
- Handy boilerplate structure for pages and components
- Ready for local development and production build

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - application routes and layout
- `components/` - reusable UI components
- `public/` - static assets
- `styles/` - global and theme styles

## Usage

- Update theme settings in your layout or theme provider.
- Add new pages under `app/`.
- Add components under `components/`.
- Customize styles in `styles/`.

## Build

Create an optimized production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Customization

- Change the home page by editing `app/page.js`.
- Add new theme variants in your CSS or theme provider.
- Extend components and layouts to fit your design system.

## Notes

This repo is intended as a theme boilerplate for Next.js projects. Use it as a base for your own theme-driven applications.
